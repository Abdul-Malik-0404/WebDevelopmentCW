Home Page:

The Home Page is the main entry point of the website, providing an introduction and guiding users to key sections. It features a clear navigation menu, a brief welcome message, and visually engaging elements to enhance user experience. Designed for easy access, it includes links to important pages and interactive features for seamless navigation. The layout ensures a clean and structured presentation, making it simple for visitors to explore the website.

Visual Design - The home page features our mission statement and tiles linking to our content pages. It is also the first page where the header and footer are visible.

Content - The header, which appears at the top of the webpage contains our logo on the far left side and the navigation bar on the right side. The navigation bar links to all the important pages of the website. The footer, which appears at the bottom of the page, includes the names of the page editors and content pages, making it easy for users to navigate the website.

The CSS file for the home page is named home.css. It contains all the styles applied to the home page, except for the header and footer. The CSS file for the header and footer are named global.css. It contains all the styles applied for all the elements in the header and footer. It is linked to all the pages except the splash screen.

Table Page:

The Table Page presents key conservation data in a structured and visually engaging format, allowing users to compare environmental statistics across different regions. It provides insights into deforestation, endangered species, protected land, and sustainability efforts.

Visual Design - The page features a centered title and description, followed by a well-organized table. The table uses alternating row colors for readability, a subtle shadow for depth, and hover effects to enhance user interaction. Icons next to column headers provide a quick visual reference for each category.

Content - The table highlights important data on five major regions, showcasing forest loss rates, the number of endangered species, the percentage of protected land, and an intuitive sustainability score. This structured presentation makes it easier for users to interpret the information and recognize the urgency of conservation efforts. Note that this is fictional information.

The CSS file for this page, table.css, ensures a professional and responsive design, maintaining consistency with the website's overall look.

Content Page:

The "Combating Biodiversity Loss" page educates visitors on the critical issue of biodiversity decline, outlining its causes, effects, and potential solutions. The page is structured into clear sections, each providing concise yet informative content supported by visuals and engaging design elements.

Visual Design - The page features a clean and structured layout with a readable font and a balanced color scheme. A featured image at the top enhances visual appeal, while section headings and bullet points improve readability.

Content - The page covers key topics such as deforestation, pollution, climate change, and overexploitation. It explains the consequences of biodiversity loss and highlights actionable solutions like reforestation, marine conservation, and legal protections. A prominent call-to-action button encourages users to take part in conservation efforts. Note that this is fictional information.

The CSS file for this page, content_ST2.css, ensures a professional and visually cohesive design, maintaining consistency with the rest of the website.

The JavaScript file for the page, content_ST2.js, enhances user experience by providing a "Back to Top" button functionality. This allows users to quickly return to the top of the page with a smooth scrolling effect.
